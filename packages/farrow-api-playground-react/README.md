# farrow-api-playground-react

## Development

Develop with watch mode

```bash
pnpm run build --watch
```

## Release

Build

```bash
pnpm run build
```

Bump version

```bash
npm version patch/miror/major
```

Submit version commit

```bash
git add .
git commit -m "release: x.x.x"
git push
```

Publish to npm

```bash
npm publish
```
